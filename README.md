This project is a simple command-line based ATM (Automated Teller Machine) simulation written in Python. It allows users to perform basic banking operations such as checking balance, depositing money, withdrawing funds, and viewing transaction history.

🚀 Features
💰 Check current account balance
➕ Deposit money securely
➖ Withdraw money with balance validation
📄 View transaction statement/history
❌ Error handling for invalid inputs and insufficient balance
🔁 Continuous menu-driven interface
🛠️ Technologies Used
Python 3
Object-Oriented Programming (OOP)
📂 Project Structure
ATM class:
Handles all banking operations
Stores balance and transaction history
Main loop:
Provides interactive menu for user actions
▶️ How to Run
Make sure Python 3 is installed on your system

Clone this repository:

git clone https://github.com/your-username/your-repo-name.git

Navigate to the project folder:

cd your-repo-name

Run the program:

python atm.py
📌 Example Operations
Start with an initial balance of ₹1000
Deposit any positive amount
Withdraw money if sufficient balance is available
View all past transactions in order
⚠️ Limitations
No authentication system (PIN/password)
Data is not stored permanently (resets after program ends)
Command-line interface only (no GUI)
